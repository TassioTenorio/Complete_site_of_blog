# API Blog

![Diagram blog](./docs/blog-arch.png)

## Create virtual env

```ssh 
pip install virtualenv

python3 -m venv .

source bin/activate

deactivate
```


## Install libs

```ssh
pip3 install Flask > requirements.txt
```


## Execute docker compose
```ssh 
docker-compose up -d
```

## View Containers
```ssh
docker ps
```
